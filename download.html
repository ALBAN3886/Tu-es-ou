📋 Comment installer l'APK Android (3 étapes)

1
Téléchargez l'APK
Appuyez sur le bouton vert ci-dessus. Le fichier trackteam.apk sera téléchargé dans votre dossier Téléchargements.
2
Autorisez les sources inconnues
Allez dans Paramètres → Sécurité → Sources inconnues et activez l'option. Sur Android 8+, l'autorisation est demandée automatiquement à l'installation.
3
Installez et lancez
Ouvrez le fichier APK téléchargé, appuyez sur Installer. L'application apparaît dans votre liste d'apps. Lancez-la et connectez-vous avec votre code entreprise.
Questions fréquentes
L'application est-elle gratuite ? ▼
Oui, TrackTeam est entièrement gratuit. Aucun abonnement, aucune limite d'employés.
L'APK est-il sécurisé ? ▼
L'APK est généré à partir du même code source que la version web. Les données sont stockées dans Firebase (Google) avec chiffrement en transit. Aucune donnée n'est partagée avec des tiers.
Combien d'employés peut-on suivre simultanément ? ▼
Il n'y a pas de limite technique. Vous pouvez suivre autant d'employés que nécessaire. Les performances dépendent de votre connexion internet.
L'employé peut-il voir ma position ? ▼
Non. L'employé voit uniquement sa propre position sur sa carte. Seul le patron (dashboard) voit la position de tous les employés.
Ça fonctionne sans internet ? ▼
Non, une connexion internet (WiFi ou données mobiles) est nécessaire pour envoyer les positions en temps réel vers Firebase.
L'employé peut-il arrêter le partage ? ▼
Oui, l'employé peut appuyer sur "Arrêter le partage" à tout moment depuis son écran. Le patron le verra passer en statut "Hors ligne".
